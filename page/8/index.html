bubble sort

  • bubble_sort(거품 정렬)

    • Comparson sort의 한 종류
    • 두 수를 비교해서 정렬하는 방법
    • 마지막 전 자료와 마지막 자료를 비교하여 교환하면서 자료를 정렬
    • 한번 순회가 끝나면 가장 큰 자료가 맨 뒤로 이동
    • 2번째 순회에서는 맨 끝에 잇는 자료는 정렬에서 제외되는 방식
    • 순회할 때마다 정렬에서 제외되는 데이터가 하나씩 늘어남

  • python에서의 두 변수 값 바꾸기(swap)

    • 보통 임시로 저장할 변수를 이용하여 바꿈

    • python에서는 한 줄로 바꾸기 가능

function

  • routine, subroutine, proceduce, function

  1. 함수 정의

    • 전역 변수(global variable) : global frame에 쌓인다.
    • 지역 변수(local variable) : 함수가 호출되고 stack frame이 생길때 생기고 함수가 끝나면 사라진다.
    • 함수 선언 시, 들여쓰기 공백은 4칸으로 4의 배수로만 띄워주면 문제없다.

  2. 함수 호출

    • 함수가 호출되는 시점이 바로 stack frame이 생성되는 시점이다.

  3. namespace

    • 변수의 유효 범위를 namespace 기반으로 결정

    • global문을 사용하여 func() 내에서 전역 변수 수정 가능

    • namespace 개념을 통해 ‘이름 공간’에서 가장 가까운 변수에 접근

    • nonlocal문을 사용하면 함수에서 가장 가까운 해당 변수에 접근할 수 있다.
    • nonlocal문으로 지정해주지 않아도 가장 가까운 해당 변수에 접근한다.
    • 가장 가까운 변수에 접근하게 되면 그 이상의 공간 변수로 접근하지 않는다.

operator

  1. 논리 연산자

    • AND : 모두 참일 때 빼고는 다 거짓

    • OR : 모두 거짓일 때 빼고는 다 참

    • 맨 마지막에 참조하는 객체를 반환

    • [1, 2] or [] 에서는 이미 [1, 2]에서 참 거짓 판단이 끝났으므로, 뒤 []까지 보지 않음
    • [1, 2] and []에서는 [1, 2]가 참이면 뒤에까지 봐야하므로 뒤까지 보고 판단
    • [] and [1, 2]일 경우 이미 앞에서 거짓이므로 거짓 판단
    • [1, 2] and [3, 4]이면 앞이 참이고 뒤에도 참이므로 뒤를 출력(마지막 참조 객체)

  2. 비트 연산자

    • AND : &

    • OR : |

    • XOR(EXCLUSIVE OR)

    • 서로 다르면 참, 서로 같으면 거짓

    • 비트 이동 연산자(Shift Operator)

    • 꺾여있는 방향으로 이동

  3. 산술 연산자

  • +, -, /, //, *… 등등

    • 실수형 나누기

    • 정수형 나누기

    • 제곱 표현

    • 나머지 구하기

dictionary

  • key와 value가 한 쌍
  • hash로 구현(대응 관계)

  1. Basic

    • key:value로 입력

    • 중괄호 사용

    • 새로운 값을 삽입

    • 기존 값 변경이 가능

    • 변경이 불가능한 객체를 key로 사용

    • key 값은 변경이 가능한 객체로 사용 불가능(unhashable)

  2. 관련 함수들

    1) search 계열

    - get() : key로 value 얻기
- key가 없는 값을 불러오면 에러가 발생하지만, get()을 사용하면 None을 반환

    2) delete 계열

    - pop() : 원하는 key의 value를 반환하고 삭제
- popitems() : 임의의 key와 value를 반환하고 삭제

    3) insert 계열

    - update() : key가 있으면 원하는 value로 바꾸고 key가 없으면 추가
- setdefault() : 원하는 key의 value를 찾아 반환

    4) view 객체 사용

set

  • 집합을 표현하기 위해서 만들어진 자료구조
  • 중복의 데이터를 허용하지 않음(집합 표현)

  1. Basic

    • list는 중복의 데이터를 허용한다.

    • set은 중복의 데이터를 허용하지 않는다.

    • 문자열을 입력하여 만들 수도 있다.

    • 집합 표현

      • union() : 합집합
        • ‘OR’ 연산자도 가능
      • intersection() : 교집합
        • ‘AND’ 연산자도 가능
      • difference() : 차집합
        • ‘-‘ 연산으로도 가능
      • symmetric_difference() : 대칭 차집합
        • 교집합을 뺀 나머지
        • ‘XOR’ 연산자도 가능

  2. 관련 함수들

    • add() : 요소 값 추가

      • 중복의 데이터를 추가하면 들어가지 않는다.

    • update() : 요소 값 여러 개 추가

      • 중복의 데이터는 추가되지 않는다.

    • remove() : 원하는 요소 값 제거

    • pop() : 임의의 요소 값을 반환한 후, 삭제

    • difference() vs difference_update()

      • difference()는 기존 리스트가 바뀌지 않음
      • difference_update()는 기존 리스트가 바뀜

list

  • python에서 list는 변경 가능한 객체(mutable)
  • 자료형에 상관없이 만들 수 있다.

  1. Basic

    • C 언어에서의 정의

    • python에서의 정의

    • 인덱싱이 가능하다.

    • python에서는 음수 인덱싱이 가능하며, 뒤쪽부터 역순으로 접근한다.

    • 연산도 가능하다.

    • 슬라이싱도 가능하다([start(포함):end(미포함)]).

  2. 관련 함수들

    • 자료 구조는 크게 3가지로 나눈다.
      1) insert(삽입)
      2) search(검색)
      3) delete(삭제)

    1) insert 계열

    • append() : 리스트 끝에 요소 값 추가

    • insert() : 원하는 자리를 지정하여 요소 값 추가

    • extend() : 인자를 리스트로 받아 기존 리스트를 확장

    2) search 계열

    • 인덱싱

    • index() : 원하는 요소 값의 위치를 반환

    3) delete 계열

    • remove() : 원하는 요소 값 제거

      • 값이 중복될 경우, 첫번째 값만 제거
      • 반복해서 제거 가능

    • pop() : 요소 값 가져오기

      • 인자를 넣지 않으면 요소 끝 자리에 있는 값을 가져옴
      • 리스트에서는 삭제

    • 정렬 함수

      • sort() : 오름차순 정렬
      • sort(reverse = True) : 내림차순 정렬
      • sorted(list) : 오름차순 정렬(기존 리스트 차순을 변경하고 싶지 않을 때 사용)

string

  • python에서 문자열은 변경이 불가능한 객체(immutable)

  1. Basic

    • python의 문자열은 ‘ ‘ 와 “ “ 모두 사용하며, 같이 사용할 수도 있다.

    • 문자열 안에 ‘와 ‘’를 같이 쓰고 싶은 경우, ‘\‘와 같이 쓰면 문자 취급을 할 수 있다.

    • “”” “”” 사이에 있는 모든 코드를 다 문자 취급할 수 있다(multi line).

    • 문자열 연산도 가능하다.

    • 문자열 인덱싱 및 길이 출력이 가능하다.

    • 인덱싱 범위가 벗어난 경우, 에러가 발생한다.

  2. Slicing(슬라이싱)

    • [start(포함) : end(미포함)]

  3. Format(포맷)

    • 형식

    • ‘{인덱스:<길이}’.format(값) : 부등호 방향으로 정렬 후, 공백으로 채우기

    • {인덱스:0의 개수.자릿수f}’.format(숫자): 자릿수 지정 가능

  4. 관련 함수들

    • isdigit() : 문자열이 숫자인지 아닌지를 True 또는 False로 반환

    • isalpha() : 문자열이 문자인지 아닌지를 True 또는 False로 반환

    • count() : 해당 문자 개수 반환

    • strip() : 문자열 기준으로 양쪽 공백 지우기

    • lstrip() : 문자열 기준으로 왼쪽에 있는 공백 지우기

    • rstrip() : 문자열 기준으로 오른쪽에 있는 공백 지우기

    • replace() : 문자열 바꾸기

    • split() : 문자열 나누기

    • 괄호 안에 아무런 값도 넣지 않으면 공백(스페이스, 탭, 엔터 등) 기준으로 나눔

    • join() : 문자열 삽입

    • 괄호 안에 아무런 값을 넣지 않아도 가능
