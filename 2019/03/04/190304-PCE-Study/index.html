2019.03.04 Python and CE Study

String

  • python에서 문자열은 변경이 불가능한 객체(immutable)

  1. Basic

    • python의 문자열에서는 ‘ ‘와 “ “ 모두 사용하며, 같이 사용할 수도 있다.

      a = 'abcde'
print(a)
>> abcde

a = "abcde"
print(a)
>> abcde

    • 문자열 안에 ‘와 “를 같이 쓰고 싶은 경우, ‘\’ 와 같이 쓰면 문자 취급을 할 수 있다.

      b = "I said, 'I can do it!'"
print(b)
>> I said, 'I can do it!'

b = "I said, \'I can do it!\'"
print(b)
>> I said, 'I can do it!'
    • “”” “”” 사이에 있는 모든 코드를 문자 취급할 수 있다(multi line).
      a = """
hello world!
hello world!!
hello world!!!
"""
print(a)
>> hello world!
   hello world!!
   hello world!!!

    • 문자열 연산도 가능하다.

      a = 'abcd'
b = 'efgh'
print(a+b)
>> abcdefgh

print('*'*30)
>> ******************************
    • 문자열도 인덱싱 및 길이 출력이 가능하다.

    • 인덱싱 범위가 벗어날 경우 에러가 발생한다.

      c = "abcde"
c[0]
>> 'a'

c[1]
>> 'b'

c[10]
>> ---------------------------------------------------------------------------
    IndexError                                Traceback (most recent call last)
    <ipython-input-11-ea0c20b3b68b> in <module>
    ----> 1 c[10]

    IndexError: string index out of range

a = 'abcd'
len(a)
>> 4

  2. Slicing(슬라이싱)

    • [start(포함) : end(미포함)]

      a = "abcdefghijklmnopqrstuvwxyz"
a[1:10]
>> 'bcdefghij'

a[:10]
>> 'abcdefghij'

a[6:]
>> 'ghijklmnopqrstuvwxyz'

a = "I love python"
a = a[:2] + "like" + a[6:]
print(a)
>> I like python

  3. Format(포맷)

    • 다양한 문법

      'I like the number %d and color %s' % (2, 'black')
>> 'I like the number 2 and color black'

name1 = 'greg'
name2 = 'john'
'hi {1} my name is {0}'.format(name1, name2)
>> 'hi john my name is greg'

'hi {} my name is {}'.format(name1, name2)
>> 'hi greg my name is john'

# python3.6이상에서만 지원되는 문법
a = 10
b = "abcde"
f'value of a {a} and b is {b}'
>> 'value of a 10 and b is abcde'

    • ‘{인덱스:<길이}’.format(값) : 부등호 방향으로 정렬 후, 공백으로 채우기

      '{0:>10}'.format(7)
>> '         7'

'{0:>1}'.format(7)
>> '7'

'{0:<10}'.format(7)
>> '7         '

'{:<20}'.format(7)
>> '7                   '

f'{"koo":<20}'
>> 'koo                 '
