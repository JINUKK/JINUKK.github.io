set

2019-03-04
python, set
  • 집합을 표현하기 위해서 만들어진 자료구조
  • 중복의 데이터를 허용하지 않음(집합 표현)

  1. Basic

    • list는 중복의 데이터를 허용한다.

    • set은 중복의 데이터를 허용하지 않는다.

    • 문자열을 입력하여 만들 수도 있다.

    • 집합 표현

      • union() : 합집합
        • ‘OR’ 연산자도 가능
      • intersection() : 교집합
        • ‘AND’ 연산자도 가능
      • difference() : 차집합
        • ‘-‘ 연산으로도 가능
      • symmetric_difference() : 대칭 차집합
        • 교집합을 뺀 나머지
        • ‘XOR’ 연산자도 가능

  2. 관련 함수들

    • add() : 요소 값 추가

      • 중복의 데이터를 추가하면 들어가지 않는다.

    • update() : 요소 값 여러 개 추가

      • 중복의 데이터는 추가되지 않는다.

    • remove() : 원하는 요소 값 제거

    • pop() : 임의의 요소 값을 반환한 후, 삭제

    • difference() vs difference_update()

      • difference()는 기존 리스트가 바뀌지 않음
      • difference_update()는 기존 리스트가 바뀜
