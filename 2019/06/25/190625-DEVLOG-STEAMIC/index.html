일단은! 이것을 우회(?)하려고 여러가지 방법을 쓰다가 jsonp라는 것을 알게 되었다.

ajax 호출은 동일한 도메인에서만 호출이 가능하다는 점을 알게 되었다.

생각해보니 실습 프로젝트에서는 ajax를 같은 도메인 안에서만 적용했었다.

jsonp는 기존 json 방식에서 function 형태로 바꾼 것이라고 한다.
